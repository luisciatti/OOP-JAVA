A Java application developed using **Object-Oriented Programming (OOP)** as part of coursework for a **Bachelor’s Degree in Computer Science (BCC)**.

The project follows a **modular architecture inspired by MVC (Model–View–Controller)** to separate responsibilities between system components and improve maintainability.

All classes are documented using **JavaDoc**.
